# LIBRARY MANAGEMENT SYSTEM

## Status
This project is currently under development. Features and APIs may change.

## Description
The library system is currently under development and will be deployed on AWS or K8S in the near future. The project simulates the management of books and user orders in real-world scenarios. Instead of having to go to the library to complete cumbersome processes, users can use the software to perform online operations such as ordering books, borrowing books, etc.

## Tech stack
- Go
- Python
- Docker
- PostgreSQL
- Kubernetes
- gRPC/REST
